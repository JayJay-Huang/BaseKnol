# 關於安全防護

需要情況 : 
> 會儲存個人可識別資訊(PII)。

HTTPS : 建立安全網站，身分驗證機制的基礎，並把重心放在第三方驗證。


HTTPS

> 不提供身分驗證，但是身分驗證基礎。
> HTTPS協定建立在擁有「SSL ( 工鑰憑證 ) 」的伺服器上。
> 目前SSL標準格式，稱為 : X.509。
> 認證機構( CA ) ，會製作「受信根憑證」供瀏覽器使用。


# 取得憑證

自己製作、從非營利CA取得、向營利CA購買。
